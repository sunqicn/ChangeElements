## 插件简介

这个插件旨在增强网页的可定制性，通过隐藏或转换网页上的特定内容，帮助用户根据个人需求调整浏览体验。

### 功能

1. **隐藏 Logo**: 取消页面上显示的 Logo 图像。
2. **隐藏标题**: 隐藏页面上的标题元素（例如 `<h1>`, `<h2>` 等）。
3. **标题转换**: 将标题元素（`<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`）转换为普通段落（`<p>`），并将粗体元素 `<b>` 转换为普通段落。
4. **取消固定内容**: 隐藏那些使用 `position: fixed` 定位的页面元素，例如固定在页面顶部的导航栏。

### 安装

1. 下载插件文件。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/` 页面。
3. 开启开发者模式。
4. 点击“加载已解压的扩展程序”，选择插件文件夹。
5. 插件将被添加到浏览器中，并可在扩展程序列表中找到。

### 使用

1. 点击浏览器工具栏中的插件图标打开设置页面。
2. 通过勾选相应的复选框来启用或禁用功能。
3. 设置会自动保存并立即应用到当前网页。
